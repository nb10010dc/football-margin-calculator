# 足彩抽水计算器

一个用于计算胜平负赔率抽水、返还率、隐含概率和投注分配比例的静态网页工具。
目的：以最低损耗刷流水，或是用于赠送彩金提现

## 功能

- 输入胜 / 平 / 负十进制赔率
- 自动换算对应胜率
- 计算庄家抽水程度
- 计算返还率
- 计算去水后的公平概率和公平赔率
- 根据任一投注额自动匹配另外两个选项
- 根据总投注额自动分配胜 / 平 / 负投注额
- 根据“想拿回现金”反推所需投注额
- 显示理论净损耗、彩金兑换率、每 1000 流水成本

## 使用方式

打开网页后，输入主胜、平局、客胜三项赔率即可自动计算。

也可以输入：

- 任一赛果投注额
- 总投注额
- 想拿回现金金额

系统会自动计算对应的胜平负投注比例。

## 部署

本项目是纯静态网页，无需服务器和数据库。

GitHub Pages 部署方式：

1. 上传 `index.html` 到仓库根目录
2. 进入仓库 `Settings`
3. 打开 `Pages`
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main`
6. Folder 选择 `/ root`
7. 保存后等待 GitHub Pages 生成访问地址

## 注意

本工具仅用于赔率、概率和抽水结构的数学计算，不构成投注建议。请遵守所在地法律法规，理性使用。
